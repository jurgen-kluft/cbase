# xbase

Cross platform base library

- xbase
  - types (s8/u8 .. s64/u64, f32/f64, uchar/../uchar32)
  - allocator
  - binary search
  - bitfield
  - buffer
  - console
  - debug (assert)
  - endian
  - integer
  - limits
  - log
  - printf/sprintf
  - random (interface)
  - singleton
  - slice
  - sort
  - tls
  - low-level string functions
  - va-list
