# cbase

Cross platform base library

- cbase
  - types (s8/u8 .. s64/u64, f32/f64, uchar/../uchar32)
  - allocator
  - binary search
  - bitfield
  - buffer / binary reader / binary writer
  - console
  - debug (assert)
  - endian
  - integer
  - limits
  - log
  - printf / sprintf
  - random (interface)
  - runes (ascii, utf-8, utf-16, utf-32) and many string manipulation functions
  - singleton
  - slice
  - sort
  - tls
  - low-level string functions
  - va-list
